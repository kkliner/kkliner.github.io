2/5/2023

I spent some time trying to figure out what flashlight I should get for scuba diving. For a while I thought the lights with focusable beams were cool until I realized that I have never once been in a scenario where that's mandatory. I ended up finding this guide and using this tool to narrow down the search, and ultimately ended up ordering this flashlight which was one sale for $34.99 with free shipping. Seems like a great deal considering dive lights can easilly sell for 2-5 times that. I like that this model has a rotary switch instead of a push button. It should be much easier to operate blind and should entirely prevent cycling through a bunch of high power settings before getting to the one you actually want to use. Keeps everyone's retinas intact. I'm slightly skeptical about build quality and longevity, but it had good reviews on Amazon so we'll see.
